# Dark-forest-fantasies
An infinite console-based survival RPG adventure. Survive days and nights in a mysterious forest, battle monsters, collect coins, and uncover dark secrets!

Dark Forest Fantasies

by markaked

An infinite text-based adventure game set in a mysterious dark forest. Play as a traveler navigating through strange encounters, battles, and discoveries in an ever-changing fantasy world.

Game Description

Dark Forest Fantasies is a console-based RPG adventure where you wake up in a mysterious forest with no memory of how you got there. Your goal is to survive as long as possible by:
- Exploring the forest and encountering various events
- Battling dangerous creatures
- Collecting coins and items
- Trading with mysterious merchants
- Making choices that affect your journey

The game features infinite gameplay with randomly generated events, ensuring no two playthroughs are exactly the same!

Features

- Character Creation**: Choose your name and gender
- Dynamic Events**: Random encounters including battles, shops, mysteries, and dialogues
- Combat System**: Turn-based battles with different enemies
- Inventory System**: Collect and use various items
- Shop System**: Buy weapons, armor, and potions
- Progress Tracking**: Day counter and statistics
- Multiple Endings**: Your choices matter!

Requirements

- Python 3.6 or higher
- No additional libraries required (uses only standard Python modules)
